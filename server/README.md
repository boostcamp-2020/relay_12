3. 아래와 동일하게 입력해주세요. 
![db_instruction(2)](./image/server_DBinst(2).png)
4. server/sql.sql 내용 복사, 입력

### server
1. download repository
2. cd /server
3. npm install
4. npm run start ( = npx nodemon app.js = node app.js)
